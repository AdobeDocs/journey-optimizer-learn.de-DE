# Konfigurieren mehrsprachiger Nachrichten

{{limited-availability}}

Mit der Funktion für mehrsprachige Nachrichten können Sie mühelos Inhalte in mehreren Sprachen innerhalb einer einzigen Kampagne oder Journey erstellen. Bei der Bearbeitung einer Kampagne oder einer Journey können Sie zwischen verschiedenen Sprachen wechseln, den gesamten Bearbeitungsvorgang optimieren und die effiziente Verwaltung mehrsprachiger Inhalte verbessern. Erfahren Sie, wie Sie mehrsprachige Nachrichten in Journey Optimizer konfigurieren.
