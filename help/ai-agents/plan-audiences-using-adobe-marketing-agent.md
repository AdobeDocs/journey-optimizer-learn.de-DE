# Planen von Audiences mit Adobe Marketing Agent for Microsoft 365 Copilot

Erfahren Sie, wie Adobe Marketing Agent for Microsoft 365 Copilot die Zielgruppenplanung direkt in Microsoft Teams vereinfacht. Mit diesem Tool können Marketing-Fachleute Zielgruppen für die Treue identifizieren, die Journey-Herkunft überprüfen und Kampagnenkandidaten validieren, ohne die Plattform zu wechseln oder Anfragen an die Betriebs-Teams zu stellen.

In diesem Video wird der Prozess der Auswahl, Analyse und Vorbereitung von Audiences für Kampagnen veranschaulicht. Es zeigt, wie der Agent umsetzbare Einblicke generiert, zugehörige Vorschläge aufzeigt und eine Visualisierung der Herkunft zwischen Audiences und Journey bereitstellt. Marketing-Experten können Daten effizient verwalten und Workflows optimieren, um eine zielgerichtete Zielgruppenbestimmung und verbesserte Kampagnenergebnisse sicherzustellen.
