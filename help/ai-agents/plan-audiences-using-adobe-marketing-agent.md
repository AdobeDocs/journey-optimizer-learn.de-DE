# Planen von Zielgruppen mithilfe von Adobe Marketing Agent für Microsoft 365 Copilot

Erfahren Sie, wie Adobe Marketing Agent für Microsoft 365 Copilot die Zielgruppenplanung direkt in Microsoft Teams vereinfacht. Mit diesem Tool können Marketing-Fachleute treue Zielgruppen identifizieren, Journey-Beziehungen überprüfen und Kampagnenkandidaten validieren, ohne die Plattform wechseln oder Anfragen an die Betriebs-Teams stellen zu müssen.

In diesem Video wird der Prozess der Auswahl, Analyse und Vorbereitung von Zielgruppen für Kampagnen veranschaulicht. Es wird gezeigt, wie der Agent verwertbare Erkenntnisse generiert, zugehörige Vorschläge unterbreitet und eine Visualisierung der Beziehung zwischen Zielgruppen und Journeys bereitstellt. Marketing-Fachleute können Daten effizient verwalten und Workflows optimieren, um zielgerichtetes Targeting und verbesserte Kampagnenergebnisse sicherzustellen.

Sehen Sie sich das unten eingebettete Video mit einer Live-Demo von Adobe Marketing Agent in Aktion an.
