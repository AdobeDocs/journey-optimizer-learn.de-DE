# Erstellen eines Lookup-Datensatzes

Mit einer Datensatzsuche kann Adobe Journey Optimizer zur Journey-Laufzeit Referenz- oder Transaktionsdaten aus einem Adobe Experience Platform-Datensatz abrufen, ohne dass diese Daten im Echtzeit-Kundenprofil gespeichert werden müssen.

In diesem Anwendungsfall sucht die Journey dynamisch nach den entsprechenden Investitionsdetails aus einem Referenzdatensatz, wenn eine Benutzerin oder ein Benutzer ein Investitionspräferenzformular (Aktien, Anleihen oder Einlagen) einreicht, und verwendet diese Informationen zum Personalisieren einer E-Mail.
