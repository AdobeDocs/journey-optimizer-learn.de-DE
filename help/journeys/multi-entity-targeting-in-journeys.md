# Verwenden von Targeting mit mehreren Entitäten mit Federated Audiences in Journey

Erfahren Sie, wie Sie das Targeting für mehrere Entitäten verwenden können, um über profilbasierte Zielgruppen hinauszugehen und mithilfe von verbundenen Zielgruppen relevantere Kundenerlebnisse bereitzustellen.

In diesem Video erfahren Sie, wie Sie Journey nach Abonnement-ID, Konto-ID oder Auftrags-ID ansprechen können, um eine präzisere Zielgruppenaktivierung und personalisierte Interaktion zu ermöglichen, wenn eine Kundin oder ein Kunde mit mehreren Geschäftseinheiten verknüpft ist.

Weitere Informationen zu dieser Funktion finden Sie unter [Dokumentation](https://experienceleague.adobe.com/de/docs/journey-optimizer/using/orchestrate-journeys/manage-journey/supplemental-identifier){target="_blank"}.
