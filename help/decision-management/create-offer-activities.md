# Erstellen von Angebotsaktivitäten

Erfahren Sie, wie Sie in [!DNL Offer Decisioning] [Angebotsaktivitäten](https://experienceleague.adobe.com/docs/journey-optimizer/using/offer-decisioniong/create-manage-activities/create-offer-activities.html?lang=de) erstellen können. Eine Angebotsaktivität fasst Ihre Platzierungen und Sammlungen in einer Einheit zusammen, sodass eine Entscheidung getroffen werden kann, um dem Kunden das relevanteste Angebot zu unterbreiten.
