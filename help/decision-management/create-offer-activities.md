# Erstellen von Angebotsaktivitäten

Erfahren Sie, wie Sie [Angebotsaktivitäten](https://experienceleague.adobe.com/docs/journey-optimizer/using/offer-decisioniong/create-manage-activities/create-offer-activities.html?lang=de) für die Entscheidungsfindung. Eine Angebotsaktivität fasst Ihre Platzierungen und Sammlungen in einer Einheit zusammen, sodass eine Entscheidung getroffen werden kann, um dem Kunden das relevanteste Angebot zu unterbreiten.
