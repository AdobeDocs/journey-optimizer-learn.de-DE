# Verwenden von Decisioning zum Personalisieren von E-Mails

In diesem Tutorial erkunden wir einen realen Anwendungsfall für den Versand personalisierter E-Mail-Angebote mit Adobe Journey Optimizer Decisioning. Das Szenario ist einfach: Senden Sie zielgerichtete E-Mail-Angebote an Kunden auf der Grundlage ihrer Profilattribute, wie z. B. Jahreseinkommen und Treuestatus.
Während des Tutorials gehen wir Schritt für Schritt vor - von der Erstellung von Profilen in Adobe Experience Platform über die Konfiguration des E-Mail-Kanals bis hin zur Erstellung und Ausführung der Kampagne, die automatisch das relevanteste Angebot mithilfe von Decisioning auswählt.

## Voraussetzungen für dieses Tutorial

* Zugriff auf Experience Platform

* Grundlegendes zu Experience Platform-Konzepten (Profile, Audiences, Datensätze)

* Vertrautheit mit Journey Optimizer
