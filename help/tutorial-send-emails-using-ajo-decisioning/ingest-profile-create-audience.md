# Aufnehmen eines Profils und Erstellen einer Audience

In diesem Video erfahren Sie, wie Sie ein Profilschema erstellen, Kundendaten aus einer CSV-Datei aufnehmen und mithilfe von Profilattributen in Adobe Experience Platform eine Zielgruppe erstellen.

Sie erfahren, wie Sie Schlüsselfelder wie Vorname, E-Mail, Treuestatus und Jahreseinkommen definieren, echte Kundendaten hochladen und diese Daten verwenden können, um eine Audience von hochwertigen Kunden zu erstellen.
